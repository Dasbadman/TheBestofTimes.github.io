Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Optionals into the pop-up and select OK

Just Mods Assorted

Install normally

- Collection of highly configurable QOL/gameplay improvements, such as sprinting and a Fallout 4-style loot menu

BLEED

Install normally

- Overhauls how damage and force is applied, making combat quicker and harder, and making previously unviable builds such as unarmed more viable

BLEED Addendum

Install normally

- Balance changes/fixes for BLEED, see the included read-me for a list of the changes

SawyerBatty

Install normally

- An edited version of the JSawyer mod (a rebalance mod made by the lead director of FNV) made to work with TTW

Anniversary Animation Pack

Files to install normally:

  • Main File - Anniversary Anim Pack
  • Optional File - Tale of Two Wastelands - 3.2.2 Patch
    • If you do not know what to do when installing multiple files from the same mod page, read the IMPORTANT: Mod Installation Advice in Mod Organizer 2 section from the Mod Organizer 2 page

- Includes three of Hitman's previous animation mods, Subtle Camera Motion, Iron Sights Recoil Animations, and Fire Animation Variants, as well as some new previously unreleased animations

Clarity

Files to install normally:

  • Optional File - Clarity - TTW 3.2 - Dark Nights - carxt
    • Non-darker nights version available here (does not require the original)

- Removes the atmospheric tint in both FNV and FO3
