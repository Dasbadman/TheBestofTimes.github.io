Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Optionals into the pop-up and select OK

Just Mods Assorted

Install normally

- Collection of highly configurable QOL/gameplay improvements, such as sprinting and a fallout 4-style loot menu

BLEED

Install normally

- Overhauls how damage and force is applied, making combat quicker and harder, and making previously unviable builds such as unarmed more viable

BLEED Addendum

Install normally

- Balance changes/fixes for BLEED

SawyerBatty

Install normally

- An edited version of the JSawyer mod (a rebalance mod made by the lead director of FNV) made to work with TTW

Ammo Count Memory

Install normally

- Makes weapons no longer automatically reload when you switch between them

JIP Companions Command and Control

Install normally

- Changes the way companions are controlled and interacted with and adds new tactical features similar to isometric RPGs

Iron Sights Recoil Animations

Files to install normally:

  • Main File - Iron Sights Recoil Anims -TTW or WMIM

- Adds actual visual kick/muzzle flip to the iron sights fire animations

Subtle Camera Motion

Install normally

- Replaces the camera movement in every 1st person animation with more subtle motion

Faster Pip-Boy Animation

Files to install normally:

  • Main File - Faster Pip-Boy Animation (2x)

- Doubles the speed up the Pip-Boy opening animation

Functional Post Game Ending

Files to install normally:

  • Main File - Functional Post Game Ending - TTW Support

- Adds a fully functional post-game world to the game that changes based on your choices in-game

Clarity

Files to install normally:

  • Optional File - Clarity - TTW 3.2 - Dark Nights - carxt

- Removes the atmospheric tint in both FNV and FO3
