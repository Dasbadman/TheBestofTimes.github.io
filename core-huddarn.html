Creating a Separator in Mod Organizer 2

  • Right-click in the left pane of MO2 and select Create Separator
  • Input Essentials into the pop-up and select OK

lStewieAl's Tweaks

Files to install normally:

  • Main File: Stewie Tweaks
  • Optional File: Stewie Tweaks INI

NOTE: If you do not know what to do when installing multiple files from the same page through MO2, go back to the MO2 Config page and read the IMPORTANT: Mod Installation Advice in Mod Organizer 2 section


Special activation instructions:

  • Run the game and exit at the main menu
  • Double-click on Stewie Tweaks INI in the left pane of MO2
  • In the INI-Files tab, click on nvse/plugins/nvse_stewie_tweaks.ini
  • All the features of the mod can be turned on/off and tweaked here (they are all off by default) OR download my preset INI here

- Collection of many configurable quality of life improvements/bug fixes
NOTE: When the mod gets updated, you need to run the game once for the new features to show up in the INI
