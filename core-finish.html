Load Order.txt

Installation Instructions:

  • Extract loadorder.txt to Mod Organizer 2/profiles/Tale of Two Wastelands
  • Once the file is extracted, right-click in the left pane of MO2 and select All Mods -> Refresh

- Automatically ensures your load order is correct for the TTW and this guide

Finishing Notes

Congratulations! You now have TTW set up perfectly and can enjoy a completely stable/smooth experience. If you plan on adding your own mods, please reference the TTW 3.2+ Incompatibility List. If you have any problems with this guide, please do not bother the TTW team about it! Join this guide's Discord if you have any questions. Happy trails!
