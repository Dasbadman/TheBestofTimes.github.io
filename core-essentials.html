Vanilla HUD Cleaned

Installation instructions:

  • Download the main file through MO2 and begin the installation normally
  • You will be prompted with a FOMOD installer, in which you should check every option EXCEPT Clean Fonts for Darn then select Install

- Cleans up many HUD textures by up-scaling the textures and removing the noise

Consistent Pip-Boy Icons

Installation instructions:

  • Download the main file through MO2 and begin the installation normally
  • You will be prompted with a FOMOD installer, in which you should check Consistent Pip-boy Icons Plugin, then select Install
    • You can look through the FOMOD and select/deselect custom icons if you want, but the default settings will work fine

- Up-scales and fixes 1000s of icons, along with adding unique custom icons for many items that used to have generic/reused icons
