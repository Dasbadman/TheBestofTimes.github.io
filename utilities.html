Utilities

xNVSE

Installation instructions:

  1. Click the nvse_5_1_beta6.7z link to download the file
  2. From the downloaded archive, extract all files ending in .dll and the .exe (4 total files) to the game's Root folder
    • If you do not know what the game's Root folder is, read the Key Terminology section from the Initial Setup page
nvse_steam_loader.dll is still needed with the GOG version of the game.

- An updated version of NVSE, which extends the scripting capabilities of the engine and is essential for most mods

JIP LN NVSE Plugin

Installation instructions:

  1. Download the Main File - JIP LN NVSE Plugin 55.80 using the Manual Download button
    • Using the Manual Download button will be referred to as "downloading manually" from now on
  2. From the downloaded archive, extract the nvse folder to the game's Data folder
    • The files from the mod, and the rest of the mods in this section, should end up here

- NVSE extension with 800+ new functions and many engine bug fixes

NVSE plugins will be installed directly into the game's Data folder rather than through MO2 because MO2's virtual file system can sometimes fail to load .dll files.

JohnnyGuitar NVSE

Installation instructions:

  1. Download the Main File - JohnnyGuitar NVSE 3.40 manually
  2. From the downloaded archive, extract the NVSE folder to the game's Data folder

- Another NVSE extension similar to JIP LN

New Vegas Anti-Crash

Installation instructions:

  1. Download the Main File - NVAC - New Vegas Anti Crash 7.5.1.0 manually
  2. From the downloaded archive, extract the NVSE folder to the game's Data folder

- Works to greatly reduce the rate of game crashes

New Vegas Tick Fix

Installation instructions:

  1. Download the Main File - NVTF V9-Revision-2 manually
  2. From the downloaded archive, extract the NVSE folder to the game's Data folder
  3. Once the folder has been extracted, navigate to Fallout New Vegas\Data\nvse\plugins\ and open NVTF.ini in a text editor
  4. Change the following options:
    • Set bModifyDirectXBehavior to 1 (line 25)
    • Set bAlternateGTCFix to 1 (line 40)
    • Set iMaxFPSTolerance to 300 (line 51)
    • Leave the rest of the settings default

- Works to fix micro-stuttering, improve performance, and allow playing at higher framerates

Mod Limit Fix

Installation instructions:

  1. Download Main File - FNVModLimitFix 2.5 manually
  2. From the downloaded archive, extract the game's Data folder to the game's Root folder

- Raises the game's plugin limit to 255 and improves performance/load times, even if you are far below the plugin limit

OneTweak

Although this mod is on the Skyrim Nexus, it will still work fine for New Vegas provided you follow the installation instructions.

Installation instructions:

  1. Download the Main File - OneTweak 2.1.0.2 manually
  2. From the download archive, extract OneTweak.ini and OneTweak.dll from OneTweak 2.1.0.2\SKSE\plugins\ to Fallout New Vegas\Data\nvse\plugins\
  3. Once the files have been extracted, click the button at the top of MO2 and select INI Editor
  4. Select the FalloutCustom.ini tab and paste in the following:

    5. [Display]
    bFull Screen=0

- Enables borderless windowed mode for safe alt-tabbing

Testing Utilities
  1. Launch the game using the Tale of Two Wastelands option in MO2 (NOT the NVSE option)
  2. Once the game has reached the main menu, hit the ~ key (above the tab key) to open the console
  3. Enter GetNVSEVersion, and the console should print NVSE version: 5
    • If not, xNVSE is installed incorrectly
  4. Next, enter GetIsLAA, and the console should print GetIsLAA >> 2
    • If it prints >> 1, then you need to run Mod Organizer 2 as an administrator
    • If it prints an error message, then JIP LN NVSE is installed incorrectly
  5. Next, enter IsDLLLoaded NVAC, and the console should print IsDLLLoaded "nvac.dll" >> 1
    • If it prints >> 0, then NVAC is installed incorrectly
    • If it prints an error message, then JohnnyGuitar NVSE is installed incorrectly
  6. Next, enter IsDLLLoaded NVTF, and the console should print IsDLLLoaded "nvtf.dll" >> 1
    • If not, then NVTF is installed incorrectly
  7. Next, enter IsDLLLoaded mod_limit_fix, and the console should print IsDLLLoaded "mod_limit_fix.dll" >> 1
    • If not, then Mod Limit Fix is installed incorrectly
  8. Next, enter IsDLLLoaded OneTweak, and the console should print IsDLLLoaded "onetweak.dll" >> 1
    • If not, then OneTweak is installed incorrectly
