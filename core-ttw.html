In this step, we will properly install Tale of Two Wastelands through Mod Organizer 2

Running the Installer

NOTE: If you run into any errors when installing, check the FAQ to see if it has your solution

  • Make sure the game and/or Mod Organizer 2 are closed
  • Download the TTW Installer from here
    • Download the 3.2 Installer, 3.2.1 Incremental Update, and 3.2.2 Incremental Update files
  • Extract the contents of the 3.2. Installer archive to a folder called TTW Installer in Modding Folder/Tools
  • Extract the contents of both the 3.2.1 Incremental Update and 3.2.2 Incremental Update archives to the same folder
  • Right-click the TTW Install.exe and select Run As Administrator
    • NOTE: There is also a .bat file that will install TTW without encoding the audio that will take a considerably shorter amount of time. However, this is not supported by the guide because it will cause crackling in voice lines of NPCs wearing helmets and break ambient tracks
  • On the Select package to install pop-up, select TTW_v3.2_Full then Install
  • In the installer, ensure the file paths to both the Fallout 3 installation and Fallout New Vegas installation are correct
  • Set the Tale of Two Wastelands install path to Your Mod Organizer 2 Folder/mods/Tale of Two Wastelands 3.2
    • For example, here are my filepaths (YOUR FILEPATHS WILL NOT BE EXACTLY THE SAME AS MINE, THIS IS JUST AN EXAMPLE)
    • Triple-check to be 100% sure you have the correct filepaths
  • Select Install
    • The install will take a very long time (can be anywhere from 2-12 hours depending on your CPU, took 3 hours on my i5 7500). It is recommended to leave your computer be and have an CPU-intensive applications closed during this time as to not overheat your CPU and have the TTW installation fail
  • Once the installation has completed, you should see this:
  • Exit the installer and then immediately re-run it as an administrator
  • This time, select TTW_v3.2_to_v3.2.1_Update and then Install
  • In the installer, ensure that the filepath for New Vegas is correct, and that the filepath for Tale of Two Wastelands v3.2 is pointed to the same folder you set as the install location when installing 3.2 (Your Mod Organizer 2 Folder/mods/Tale of Two Wastelands 3.2)
  • Set the filepath for Tale of Two Wastelands v3.2.1 to the exact same folder as Tale of Two Wastelands 3.2 (Your Mod Organizer 2 Folder/mods/Tale of Two Wastelands 3.2)
    • For example, here are my filepaths (YOUR FILEPATHS WILL NOT BE EXACTLY THE SAME AS MINE, THIS IS JUST AN EXAMPLE)
  • Select Install
  • The installation should only take a few seconds, then you should see this installation completed message:
  • Exit the installer and then immediately re-run it as an administrator one more time
  • This time, select TTW_v3.2.1_to_v3.2.2_Update and then Install
  • In the installer, ensure that the filepath for New Vegas is correct, and that the filepath for Tale of Two Wastelands v3.2.1 is pointed to the same folder you set as the install location when installing 3.2.1 (Your Mod Organizer 2 Folder/mods/Tale of Two Wastelands 3.2)
  • Set the filepath for Tale of Two Wastelands v3.2.1 to the exact same folder as Tale of Two Wastelands 3.2 (Your Mod Organizer 2 Folder/mods/Tale of Two Wastelands 3.2)
    • For example, here are my filepaths (YOUR FILEPATHS WILL NOT BE EXACTLY THE SAME AS MINE, THIS IS JUST AN EXAMPLE)
  • Select Install
  • The installation should only take a few seconds, then you should see the installation completed message and you can exit the installer
  • If you wish, you can now safely uninstall Fallout 3

Activating TTW in MO2

  • Launch Mod Organizer 2
  • Make sure the Tale of Two Wastelands profile is active
  • If you installed TTW correctly with the correct filepaths, you should see the Tale of Two Wastelands 3.2 mod in the left pane of MO2
    • ×
  • Click the checkbox next to it, and the right pane should fill up with plugins
    • These plugins will automatically sort into the correct order and you will be unable to move them
  • Right-click on Tale of Two Wastelands 3.2 in the left pane of MO2 and select Open in Explorer
  • Navigate to nvse/plugins and delete the textinput folder, jip_nvse.dll, and jip_nvse.ini
    • TTW comes with an outdated version of JIP, we will installed the current version in the next step
  • Click the button on the top bar of MO2 and select INI Editor
  • Navigate to the FalloutCustom.ini tab, which should be blank
  • Paste the following into the box:
    • [Archive]
    • SArchiveList=Fallout - Invalidation.bsa, Fallout - Textures.bsa, Fallout - Textures2.bsa, Fallout - Meshes.bsa, Fallout - Meshes2.bsa, Fallout - Voices1.bsa, Fallout - Sound.bsa, Fallout - Misc.bsa

    • [General]
    • bPreemptivelyUnloadCells=1
    • bUseThreadedAI=1
    • INumHWThreads=4

    • [Audio]
    • iAudioCacheSize=8192
    • iMaxSizeForCachedSound=1024

    • [BackgroundLoad]
    • bBackgroundCellLoads=1
    • bSelectivePurgeUnusedOnFastTravel=1

    • [Grass]
    • fGrassStartFadeDistance=7000

    • [Display]
    • bFull Screen=1
    • iPresentInterval=1

    • [Controls]
    • fForegroundMouseAccelBase=0
    • fForegroundMouseAccelTop=0
    • fForegroundMouseBase=0
    • fForegroundMouseMult=0
    • fXenonVertLookSpeed=1200.0000
    • fXenonHorizLookSpeed=1500.0000
  • Change INumHWThreads to the number of threads your CPU has
    • If you do not know how many threads your CPU has, use CPU-Z to find out
  • iPresentInterval can be set to 0 if you use G/Free-sync, or want to use GPU Driver v-sync
  • fXenonVertLookSpeed and fXenonHorizLookSpeed can be removed if you do not run the game in a 16:9 aspect ratio
  • Repeat for the Testing profile
Tweak Effect
bPreemptivelyUnloadCells=1 Helps prevent Out of Memory crashes when changing cells
bUseThreadedAI=1 Allows AI to use threads to increase performance
INumHWThreads=4 Allows the game to better utilize the CPU
iAudioCacheSize=8192
iMaxSizeForCachedSound=1024 		Increases the audio cache to prevent stuttering and audio not playing
bBackgroundCellLoads=1 Streams in assets to prevent stuttering at the cost of objects sometimes suddenly popping in
bSelectivePurgeUnusedOnFastTravel=1 Helps prevent Out of Memory crashes when changing cells
iPresentInterval=1 Enables V-Sync
bFull Screen=1 Ensures the game will run in fullscreen
fForegroundMouseAccelBase=0
fForegroundMouseAccelTop=0
fForegroundMouseBase=0
fForegroundMouseMult=0 		Disables mouse acceleration
fXenonVertLookSpeed=1200.0000
fXenonHorizLookSpeed=1500.0000 		Sets the mouse sensitivity of the x and y axes equal to each other
fGrassStartFadeDistance = 7000 Increases the distance at which grass disappears from 3 cells to 5 cells
  • It is not recommended to change any other .INI settings other than the ones in the guide, especially not the following:
    • uGridsToLoad=5: Changes the amount of cells loaded at once. When changed, it can cause scripts to run when they aren't supposed to, will ruin performance, and will break your game if changed on an existing save
    • bMultiThreadAudio=0: Can cause crashes when set to 1
    • INumHavokThreads=1: Any changes to it are placebo in regards to performance, can cause instability if set over 2
    • bUseThreadedBlood/Morpher/TempEffects/ParticleSystem=0: Setting any of these to 1 is placebo and will not affect performance
