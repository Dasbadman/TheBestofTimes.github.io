TTW Wrist Fix

Installation instructions:

  • Follow the Google Drive link and download the mod using the download button pictured below
  • Install the mod normally through MO2

If you do not know how to download mods that do not have a Mod Manager Download button, read the Mod Installation Advice section from the MO2 page

- Fixes a bug in Tale of Two Wastelands that would sometimes cause enemies to hang in the air by their wrists when killed

lStewieAl's Tweaks Custom INI

Installation instructions:

  • Download the mod using the download button pictured below
  • Finish the install normally through MO2

- My personal INI for Stewie's Tweaks that enables many of the Bug Fixes/QOL Improvements of the mod. Gameplay changes will be covered in the Gameplay section

As these are just my personal preferences, my INI can be exchanged for any other preset INI (such as Axonis's), or you can download the Optional File - Stewie Tweaks INI 2.1 from the Stewie's Tweaks page which has everything disabled, and go through the INI and tweak all the features to your own liking. You will need some form of an INI in order for the mod to fully function.

Vanilla HUD Cleaned

Installation instructions:

  1. Download the Main File - Vanilla Hud Cleaned 0.381 and begin the installation normally through MO2
  2. You will be prompted with a FOMOD installer, in which you should select everything EXCEPT Clean Fonts for Darn then select Install

- Hand-made up-scaled/cleaned recreations of many vanilla HUD elements

Consistent Pip-Boy Icons

Installation instructions:

  1. Download both the Main File - Consistent Pip-boy Icons 3.605 and the Update File - Consistent Pip-boy Icons Update 3.607 normally through MO2
  2. Begin the installation of the main file first, and you will be prompted with a FOMOD installer in which you should just select Install
  3. Once the main file has been installed, install the update file normally
If you do not know what to do when installing multiple mods from the same page, read the Mod Installation Advice section from the MO2 page

- Up-scaled/fixed versions of 1000s of icons all remade from scratch
