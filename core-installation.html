These steps will take you over how to do fresh installations of Fallout New Vegas and Fallout 3, which is required for this guide

Installation Parameters:

- Fallout New Vegas, Fallout 3, and all modding tools must be installed outside all default Windows folders (Program Files, Program Files (x86), Desktop, and Documents for example)

  • This is to avoid issues with Windows UAC (User Account Control) from blocking certain aspects of different tools

- Fallout New Vegas and Mod Organizer 2 must be installed on the same drive, preferably on an SSD for shorter loading times and less stutter while loading assets

  • Fallout 3 can be installed on any drive (still outside of the default Windows folders), because it can be uninstalled after installing TTW

- You will need at least 22GB gigabytes of free drive space for just the FNV, FO3, and the TTW Installer. The total size of the entire guide is 25.5 gigabytes (including New Vegas, but excluding Fallout 3 since it can be uninstalled after installing TTW)

Common Terminology:

  • Root folder: Where you have installed Fallout New Vegas
    • For Steam it is Steam/steamapps/common/Fallout New Vegas
    • For GOG it is GOG Galaxy/Games/Fallout New Vegas (unless you manually changed the installation path)
  • Data folder: The Data folder that's located in your Root folder
    • For Steam it is Steam/steamapps/common/Fallout New Vegas/Data
    • For GOG it is GOG Galaxy/Games/Fallout New Vegas/Data
  • MO2: Mod Organizer 2 (will be installed shortly)
  • TTW: Tale of Two Wastelands (will be installed shortly)

Updating Audio Drivers

  • Windows search for Device Manager
  • In the device manager, expand the Audio inputs and outputs section, right-click on your primary audio device, and select Update driver
  • From the pop-up, select Search automatically for updated driver software
  • If the search reports that you already have the latest drivers, you can exit the window
  • If it says your drivers are outdated, let it update your drivers to the latest version

Installing Fallout New Vegas:

    For Steam:

    NOTE: If you already have a Steam library folder set up outside of the default Windows folders, you can simply install FNV and FO3 to that library folder

  • Launch Steam and select Steam -> Settings
  • Go to the Downloads tab and select Steam Library Folders
  • From the pop-up, select Add Library Folder
  • Add a folder in a location of your choice outside of the default Windows folders (preferably on an SSD)
    • If you do not have another drive to create a folder on, follow these steps (Requires Notepad++):
      • Navigate to Steam/steamapps/ and open libraryfolders.vdf with Notepad++)
      • You should see something similar to this:
      • Add a new line below the lowest filepath and follow the same format for creating a new drive path
        • If you do not see a line that starts with "1" and is followed by a filepath, just add it right below "ContentStatsID"
      • For this example, it would be something along the lines of   "3"      "E:\\Games\\Library 2 Steam"
        • Make sure the number and file path align with how many folders there already are and where you intend to create the new respectively. You also must manually create the file you are pointing it to, Steam will not create it for you
  • Exit out of the menu and go to Library
  • Select Fallout New Vegas and click Install
  • From the pop-up, select Choose location for install:
  • Select the folder you just created outside of the default Windows folders and install it
  • Repeat for Fallout 3

    • For GOG:

  • Launch GOG and select Library
  • Select Fallout New Vegas and click Install
  • Set the Install to to a location of your choice outside of the default Windows folders (preferably on an SSD) and install
  • Repeat for Fallout 3

Setting up your Fallout New Vegas Modding Folder:

Your modding folder will act as the main directory for all the tools you install, as well as any backups/screenshots/etc. you may want to keep. This folder will be referred to as the Modding folder from now on

  • Create a new folder in a location of your choice outside of the default Windows folders on the same drive as your Fallout New Vegas installation, but outside of the Root folder
  • Name it whatever you want, e.g. TTW Modding, and create two sub-folders called Backups and Tools in it
  • Copy all of the files ending in .ESM from FNV's Data folder and paste them into the Backups folder
    • Backups of the .ESMs should be made so you can recover them if you accidentally edit/delete them, since you will not be able to verify game files after installing TTW

Disabling Anti-Virus:

Any anti-virus you have, including Windows Defender, needs to have file exceptions made for your Root folder and Modding folder. It is also recommended (but not required) to completely disable your anti-virus for the duration of the guide to avoid downloaded files being flagged as dangerous.

Generating Fresh .INI Files:

For FNV:

  • Run FalloutNVLauncher.exe from the root folder
  • Click OK to both the pop-ups that say Detecting Video Hardware
    • If there aren't any pop-ups, navigate to Documents/My Games/FalloutNV and delete all the files ending in .INI then retry
  • Select Options
  • Verify the game is detecting and using your GPU properly
  • Select the High preset
    • You can select a lower one if your PC still can't handle it, but pretty much any semi-modern PC should be able to handle the High preset fine
  • Select Advanced -> Shadows and un-check Enable Shadows (New Vegas does not have real shadows, only actor shadows which are rarely visible, so you probably won't even see the difference)
    • These settings will look nearly identical to the Ultra, but with a noticeable performance increase. You can still select the Ultra preset if you really want to
  • Set Resolution to your preference (The launcher usually does not properly set it to your max resolution)
  • Click OK and then Exit

For FO3:

  • Run Fallout3Launcher.exe from the root folder
  • Click OK to both the pop-ups that say Detecting Video Hardware
    • If there aren't any pop-ups, navigate to Documents/My Games/Fallout3 and delete all the files ending in .INI then retry
  • Exit the launcher

Capping FPS

NVIDIA Profile Inspector

  • Download the NVidiaProfileInspectorDmW (non-portable) file
  • Extract the archive to a safe location
  • Run NVidiaProfileInspectorDmW.exe
  • Click Yes if there is a pop-up
  • From the Profiles: drop-down in the top left, select Fallout - New Vegas
  • In the 2. - Sync and Refresh section, set Frame Rate Limiter to 1 less than your monitor's refresh rate
    • If you do not know your monitor's refresh rate, do the following:
      • Right-click on your desktop and select Display settings
      • Scroll to the bottom of the window and select Advanced display settings
      • Make sure the monitor you play on is selected from the drop-down menu at the top of the window, and your refresh rate will be shown below Display information
        • ×
  • Click the Apply changes button in the top right then exit

  • Right-click on your desktop and select AMD Radeon Settings
  • In the Gaming tab, select the Fallout New Vegas profile
  • In the Profile Graphics tab, change the following:
  • Enable Chill (FPS)
  • Set Chill Min and Chill Max to 1 less than your monitor's refresh rate
    • If you do not know your monitor's refresh rate, do the following:
      • Right-click on your desktop and select Display settings
      • Scroll to the bottom of the window and select Advanced display settings
      • Make sure the monitor you play on is selected from the drop-down menu at the top of the window, and your refresh rate will be shown below Display information
        • ×
  • Hit the home button in the bottom left (the settings should automatically save), and make sure the word Enabled appears below the profile name
    • If not, click the three vertical dots to the right of the profile name and select Enable Profile
